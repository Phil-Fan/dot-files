# macOS 专用软件

---

**开发工具**
- [iTerm2](https://iterm2.com/) - 终端替代
  - 教程：[iTerm2 安装配置使用指南——保姆级](https://zhuanlan.zhihu.com/p/550022490)
  - 安装 Oh My Zsh：[ohmyz.sh](https://ohmyz.sh/)
  - 设置启动热键：`Preference → Keys → Create a Dedicated Hotkey Window`，设置为 `⌥⌘T`

- [Raycast](https://www.raycast.com/) - 效率启动器 (替代聚焦搜索)
  - `⌘+K` 显示
  - 参考：[能少一个是一个：我用 Raycast 替代了这些应用](https://sspai.com/post/72540)
  - 软件卸载
  - 快速打开应用：[URL Schemes | Drafts](https://docs.getdrafts.com/docs/automation/urlschemes)
  - 窗口管理
  - 参考：[Raycast 小技巧之 Quicklink 二三事](https://sspai.com/post/72951)

**系统增强**

- [Input Source Pro](https://inputsource.pro/zh-CN) - 自动切换输入法
- [ibar](https://apps.apple.com/cn/app/ibar/id6443843900) - 菜单栏隐藏
- [Pixea](https://apps.apple.com/cn/app/pixea/id1507782672) - 图片查看
- [Scroll Reverser](https://pilotmoon.com/scrollreverser/) - 反向滚动
- [Amphetamine](https://apps.apple.com/cn/app/amphetamine/id937984704) - 防止休眠

**开发辅助**
- [DevToys](https://devtoys.app/download) - 开发者瑞士军刀
- [Downie](https://www.downie.cn/) - 视频下载工具
- [Keka](https://github.com/aonez/Keka) - 压缩解压工具
- [Piclist](https://piclist.cn/) - 图床

**虚拟化 & LaTeX**
- [Parallels Desktop](https://www.parallels.cn/products/desktop/trial/) - Windows 虚拟机
- [MacTeX](https://www.tug.org/mactex/) - LaTeX 发行版
  ```shell
  brew install --cask mactex
  ```

**显示优化**
- [BetterDisplay](https://github.com/waydabber/BetterDisplay) - HiDPI 缩放解锁

---

**未签名应用信任**
```shell
xattr -cr /Applications/应用名称.app
```
