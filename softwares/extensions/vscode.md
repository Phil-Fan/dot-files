# VS Code 插件

## 基础功能

- **Chinese** - 中文语言包
- **Error Lens** - 错误提示内联显示
- **Path Intellisense** - 路径自动补全
- **Image Preview** - 图片预览
- **Prettier** - 代码格式化
- **Edit CSV** - CSV 编辑
- **Doxygen Documentation Generator** - 注释生成
- **Hungry Delete** - 智能删除

## 开发功能

- **Jupyter** - Notebook 支持
- **Remote - SSH** - 远程开发
- **LaTeX Workshop** - LaTeX 编辑支持
